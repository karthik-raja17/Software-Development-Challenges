SoftDev Challenges - Solutions Repository
=========================================

This repository contains solutions to a series of shell scripting and system administration challenges focused on text processing, log analysis, and automation using Unix tools.

Completed Challenges
--------------------

### 1. CSV Field Separator Conversion
* **Goal**: Convert CSV with quoted fields to pipe-delimited format.
* **Skills**: Text stream processing, field separator logic.

### 2. Dynamic Makefile for Monthly Builds
* **Goal**: Create a Makefile that builds different C files based on the current month.
* **Skills**: Build automation, dynamic target generation.

### 3. CommandLineFu HTML Parser
* **Goal**: Extract high-voted commands from HTML source code.
* **Skills**: Regex, command-line HTML parsing.

### 4. SSH Brute Force Detection
* **Goal**: Analyze authentication logs to detect failed login patterns.
* **Skills**: `awk`, `grep`, security log analysis.

### 5. Network Connection Geolocation
* **Goal**: Resolve IP addresses to countries for incident investigation.
* **Skills**: `geoiplookup`, subnet filtering.

### 6. Automated Directory Compression
* **Goal**: Automatically compress old, unused directories.
* **Skills**: File system traversal, `atime` checking, `tar`.

Technical Skills
----------------
* **Shell Scripting**: Bash, POSIX sh.
* **Unix Filters**: sed, awk, grep, cut, tr.
* **Automation**: Makefile development, cron-style logic.

Repository Structure
--------------------
* `challenge1/` to `challenge6/`: Individual solution directories.
* `solutions/`: Compiled scripts.
